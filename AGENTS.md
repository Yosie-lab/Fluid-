# Spacey Bloom — Agent 向けガイド

宇宙の流体を導き、星の結晶を育てる癒し系モバイル Web ゲーム。  
**単一ファイル** `index.html`（インライン CSS / JS）。バンドラなし。

## 前提

- **第一ターゲットは iPhone Safari**
- 依頼がない限り、ファイル分割・フレームワーク・ビルド導入はしない
- UI・コメントは **日本語メイン**（タイトル等は日英併記あり）
- 体験は穏やか・没入。操作説明や UI を増やしすぎない

## 構成

- すべて `index.html` 内。セクションコメントで区画されている
- CSS 変数のデザインシステムを崩さない
- iOS の AudioContext unlock（ユーザー操作後に resume）を壊さない

## 実装ルール

- 変更は該当セクションだけ。巨大ファイルなので無関係な整形・並べ替えをしない
- Canvas 2D / Web Audio の既存パターンに合わせる
- タッチ操作・フルスクリーン寄りのレイアウトを優先し、デスクトップ専用 UI に寄せない
- やり取りは日本語。確認が必要なものを除き、区切りごとに commit して GitHub へ push する

## やってほしくないこと

- 「整理のため」の複数ファイルへの分割（明示依頼があるまで）
- 依存ライブラリの追加
- 派手なエフェクトや情報密度の高い HUD への改変
